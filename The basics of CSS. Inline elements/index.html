What is inline elements?

The elements that have the inline value for the display property are known as inline elements. For example, the span, a, etc.

Default behavior of inline elements

The width and height of inline elements is calculated automatically depending on the content.

span element
span element with some text

If the inline element content doesn't fit on the one line then a browser will move it on a new line. The width will be calculated depending on maximum line length.

Also inline elements will be on the same line if there is space.

The width, height, padding, margin properties for inline elements

We can't set the width and height properties for inline elements. For example, I set 96px for the width and height for the span element.

You can add paddings, borders and margins, but the top or bottom paddings, borders and margins will end up outside the parent.
