What is block elements?

The elements that have the block value for the display property are known as block elements. For example, the div, p, etc.

Default behavior of block elements

The width of block elements fills up the horizontal space left and right on its parent element if it's there. If no, block elements fills up the viewport. The height of block elements is calculated automatically depending on the height of the content.

div element

Also block elements always start on a new line so they display one below the other.
